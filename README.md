## C++贪吃蛇

采用C++实现了一个贪吃蛇小游戏

主要分以下几个模块

- game.cpp主程序模块。包含了一些函数的调用，比如墙壁的初始化以及墙壁的显示；分数的显示；冲突按键的判断；按键判断及执行等。
- snake.cpp蛇模块。包含了蛇的初始化；头的创建；尾的删除；蛇的移动；难度的改变（通过判断蛇的长度，调用不同的速度）
- wall.cpp墙模块。包含了墙的初始化函数；以及墙的显示函数；
- food.cpp食物模块。包含设置食物函数。
